## Technology Used
- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend**: TypeScript, Express, NodeJS, MongoDB (Mongoose)
- **APIs & Authentication**: GitHub API, JWT Authentication
- **Docker and Elastic Container Service (AWS)**
- **Storage**: S3 (AWS)
- **Caching**: node-cache for Server Side Caching
- **Proxy Server**: http-proxy for Reverse Proxy Server


## Features & Usage
- **User Authentication**: Login and signup via GitHub.
- **Deployment**: Users can deploy React applications directly from their GitHub repositories.
- **Search Functionality**: Use the search bar to find GitHub repositories.
- **Real-Time Updates**: View real-time log updates during the deployment process.
- **Continuous Deployment**: Automatically deploy applications on push events from GitHub.
- **Configuration Management**: Edit site configuration and environment variables from the dashboard.
- **Custom Domain**: Set and manage custom domains for deployed websites.
- **Live Preview**: Preview the live version of hosted websites.
