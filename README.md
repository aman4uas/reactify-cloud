## Technology Used
- React
- Tailwind CSS
- TypeScript
- Express
- NodeJS
- MongoDB (Mongoose)
- GitHub API
- JWT Authentication
- Docker
- Elastic Container Service (AWS)
- S3 (AWS)
- node-cache for Server Side Caching
- http-proxy for Reverse Proxy Server


## Features
- Login/Signup from GitHub
- User can deploy React Application
- Real Time Log updates
- Autodeploy from GitHub on Push Event
- Edit site configuration and Environment Variables
- Set Custom Domain for Website
- Live Hosted Website Preview
